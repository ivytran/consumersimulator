Consumer Simulator
==================

Consumer Simulator is a 3d game made using the Unity game engine version 2020.3.30f1

==================
Instruction from Unity Hub :

Open project in Unity Hub,

Project setting => set XR Plug-In Management => OpenXR => PlayMode OpenXR runtime => Oculus
==================
XR Controller Input

Thumbstick BTN => Moving around and Turn,
Trigger BTN use => Grab grabable items
